## 1. 产品概述

### 1.1 产品定位

题库母版 APP 是一个面向职业资格考试、招聘考试、技能考试等场景的通用刷题产品底座。首版聚焦“练习、考试、资料、我的”四个主链路，帮助用户完成选科目、练题、考试、复盘、领取备考资料和管理记录的闭环。

### 1.2 核心目标

1. 用户打开 APP 后可快速进入练习。
2. 用户可以按科目、章节、真题、随机、收藏、错题等路径刷题。
3. 用户可以进行章节考试、真题考试、组卷考试，并查看考试解析。
4. 用户可以领取免费资料，试看 VIP 资料，并在开通后下载完整资料。
5. 用户可以查看练习记录、考试记录、错题、收藏、协议、反馈、上传题库等个人功能。
6. 数据结构要支持多科目、三级/四级目录、不同题型、题目媒体、资料权限和本地学习记录。

### 1.3 首版原则

1. 工具型优先：首屏直接是可操作的练习首页。
2. 数据清晰：练习进度、考试进度、正确率、错题量、记录状态要明确。
3. 路径短：常用入口从一级页 1-2 次点击可进入。
4. 题目阅读优先：刷题页减少装饰，保证题干、选项、答案和解析可读。
5. 交互可控：重置、删除、清空、交卷、重考等危险操作必须二次确认。

## 4. 核心流程

### 4.1 启动流程

```text
打开 APP
 -> P00 启动页
 -> 加载本地题库配置、科目默认值、学习记录
 -> P01 练习模式首页
```

### 4.2 练习流程

```text
P01 练习模式首页
 -> 切换科目（可选）
 -> 章节练习 / 模拟练习 / 真题练习 / 随机练习 / 收藏练习 / 错题练习
 -> 目录或设置页
 -> P05/P08B 作答
 -> 页内展示结果与解析
 -> 写入练习记录、错题、收藏、统计
```

### 4.3 章节练习目录流程

```text
P01
 -> P02 练习目录页
 -> 章节练习默认展开三级目录
 -> 点击三级目录进入 P03 小节列表
 -> 点击小节进入 P05 刷题页
```

规则：

1. 当前设计稿以“小学教师”为示例科目。
2. 题库只有三级目录时，P02 点击二级目录可直接进入 P03 或 P05，按数据是否存在下级决定。
3. 题库有四级目录时，P02 展示到三级目录，点击三级目录进入 P03 展示四级小节。
4. 章节练习和模拟真题为独立列表容器。
5. 章节/目录项右侧展示进度 `x/y`；考试目录中 `x < y`，不展示已完成绿色状态。

### 4.4 练习重置流程

```text
P02 点击“重置”
 -> P02A 底部浮窗
 -> 选择全部目录/二级目录/三级目录
 -> 父级全选、半选、未选状态联动
 -> 点击确认重置
 -> P02E 二次确认
 -> 确认后清空选中范围进度
```

勾选规则：

1. 勾选“全部目录”时，下方所有二级、三级目录全选。
2. 勾选二级目录时，该二级目录下所有三级目录全选。
3. 三级目录未满选时，二级目录和全部目录显示半选状态。
4. 所有二级目录均为全选时，全部目录显示全选。
5. 目录选择只有未选、半选、选中三种状态，不出现额外浅绿色选中状态。
6. 勾选圆圈大小统一，右侧纵向对齐。

### 4.5 考试流程

```text
P20 考试首页
 -> 章节考试 / 真题考试 / 组卷考试
 -> 目录或组卷设置
 -> P25 考试答题页
 -> P26 答题卡
 -> P27/P27A 交卷确认
 -> P28 考试解析
 -> P28A/P28B/P28C 解析详情
```

规则：

1. 考试中不展示答案和解析。
2. 未交卷记录可继续考试。
3. 交卷后不能修改答案。
4. 重考必须二次确认。
5. 考试记录标题只展示子节或试卷名称，不展示完整父级路径。

### 4.6 资料流程

```text
底部 Tab 资料
 -> P40 资料中心
 -> 免费资料：P40A
 -> 点击获取下载链接：P40B Toast

底部 Tab 资料
 -> P40 资料中心
 -> VIP 资料：P41
 -> 查看第一页
 -> 点击磨砂蒙板或获取下载链接
 -> P41A VIP 开通
 -> 支付成功/已开通
 -> P42 已解锁资料详情
```

规则：

1. 第一份资料免费。
2. VIP 资料默认只展示第一页，其余页添加毛玻璃/磨砂蒙板。
3. 点击蒙板或“获取下载链接”均进入 VIP 开通。
4. 免费资料点击“获取下载链接”后复制链接并展示 Toast。
5. 已解锁资料显示完整内容和下载入口。

### 4.7 我的与记录流程

```text
P50 我的
 -> P51 全部练习记录 / P52 全部考试记录
 -> 单条删除或删除全部
 -> 二次确认

P50 我的
 -> P53 错题练习入口
 -> P08B 错题练习页

P50 我的
 -> 登录 / VIP / 上传题库 / 意见反馈 / 关于 / 协议
```

## 5. 全局显示与交互规则

### 5.1 科目规则

1. 系统支持一个或多个科目。
2. 当前选中科目影响练习首页、考试首页、资料中心、练习目录、考试目录和记录统计。
3. P01/P20 数据面板右上角提供“切换科目”入口。
4. 科目弹窗使用胶囊按钮，一行两个，横向铺满。

6. 切换科目后刷新当前页面所有统计、入口列表和记录列表。

### 5.3 进度展示规则

1. 进度统一显示为 `x/y`。
2. `x` 不得大于 `y`。
3. 练习目录可存在 `x = y` 的完成状态，完成项可用淡绿色表示。
4. 考试目录只展示 `x/y` 且 `x < y`，不展示完成绿色条目。
5. 章节列表横向标题块可用浅蓝底表达进度，但底色不应过强。
6. 进度底色使用实色浅色值，不通过透明度降低。

### 5.4 记录标题规则

1. 练习记录、考试记录标题只显示当前子节、章节或试卷名称。
2. 不在记录标题中展示父级目录全路径。
3. 父级路径可作为辅助字段存储，必要时在详情或埋点中使用。

### 5.5 危险操作规则

以下操作必须二次确认：

1. 重置练习进度。
2. 重置考试进度。
3. 重置单个小节或真题练习。
4. 重考考试。
5. 清空错题。
6. 删除单条练习/考试记录。
7. 删除全部练习/考试记录。
8. 交卷。

确认弹窗字段：

| 字段 | 说明 |
| --- | --- |
| title | 操作标题，如“确认重置？” |
| description | 明确影响范围 |
| cancelText | 取消文案 |
| confirmText | 确认文案 |
| dangerLevel | normal/warning/danger |

1. 支持手机号验证码登录和本机号码一键登录。
2. 必须勾选用户协议和隐私协议后才能登录。
3. 用户协议进入 P61，隐私协议进入 P62。
4. 游客模式允许基础练习和考试，本地记录保存在设备。

### P59/P59A VIP

规则：

1. 从我的页或资料付费预览进入。
2. Android 支持支付宝/微信支付。
3. iOS 使用 Apple Pay / IAP。
4. 支付成功进入 P59A 或回到触发来源。
5. 资料解锁来源支付成功后可进入 P42。

### P60/P61/P62 协议页

目标：承接会员、登录、支付、关于页面中的协议阅读。

页面对应：

| 页面 | 标题 | 来源 |
| --- | --- | --- |
| P60 | 会员服务协议 | VIP 开通页、关于页 |
| P61 | 用户协议 | 登录页、关于页 |
| P62 | 隐私协议 | 登录页、关于页 |

规则：

1. 协议页为只读文本页，不展示确认按钮。
2. 页面顶部提供返回按钮。
3. 正文需支持长内容滚动。
4. 登录或支付场景中的“同意”动作在来源页完成，不在协议页完成。

## 7. 题型规则

### 7.1 题型清单

| 题型 | 示例页 | 结果态 | 作答方式 |
| --- | --- | --- | --- |
| 单选题 | QT01 | QT09 | 选择 1 个选项 |
| 多选题 | QT02 | QT10 | 选择多个选项后确认 |
| 判断题 | QT03 | QT11 | 正确/错误二选一 |
| 填空题 | QT04 | QT12 | 输入答案后确认 |
| 简答题 | QT05 | QT13 | 输入长文本后提交 |
| 材料题 | QT06 | QT14 | 公共材料 + 子题 |
| 题干图片题 | QT07 | QT15 | 题干含图片 |
| 图片加载失败 | QT08 | - | 原位置失败提示 |
| 题干多图题 | - | QT16 | 支持多图展示 |
| 解析多图题 | - | QT17 | 解析区可引用多图 |

### 7.2 通用规则

1. 图片只允许在题干或材料题公共材料中展示。
2. 选项只支持文字。
3. 图片点击可预览大图。
4. 图片加载失败时在原位置展示失败提示，不跳独立页。
5. 考试中不展示答案和解析。
6. 练习/错题/收藏/随机模式提交后在当前页显示结果态。

### 7.3 简答题评分结果

设计稿 QT13 规则：

1. 用户提交简答题后，解析位置标题显示“回答正确”或“回答错误”。
2. 标题旁显示胶囊状分数 `x/y`。
3. 当 `x / y > 60%` 时显示“回答正确”，使用绿色。
4. 当 `x / y <= 60%` 时显示“回答错误”，使用红色或警示色。
5. 结果内容顺序：

```text
正确答案：xxxx
我的答案：yyyy
解析结果：命中要点……
待补充：……
```

6. 不再单独展示“评分结果”板块。

## 12. 验收标准

### 12.1 功能验收

1. 打开 APP 后从 P00 进入 P01。
2. 底部导航为练习、考试、资料、我的。
3. P01 可切换科目，并刷新练习数据。
4. P01 可进入 P02、P06、P07、P08。
5. P02 默认展开章节练习三级目录。
6. P02 重置进度支持全选、半选、自定义选择和二次确认。
7. P03/P04 可进入 P05 刷题页。
8. P05 支持所有题型作答和页内解析。
9. P06A/P24 自选章节目录选择与重置进度选择交互一致。
10. P07 底部按钮展示当前收藏题数量。
11. P08B 错题练习页显示错误次数、最近错误，并与 P05 解析样式一致。
12. P20 可进入章节考试、真题考试、组卷考试和考试规则。
13. P21/P22 考试目录不展示已完成绿色状态。
14. P25 考试中不展示答案解析。
15. P26 答题卡可跳题和交卷。
16. P27/P27A 双交卷状态正确。
17. P28/P28A-P28C 可查看考试解析。
18. P40 资料页独立作为底部 Tab。
19. 免费资料可复制下载链接并展示 Toast。
20. VIP 资料未解锁时只可看第一页，其余页蒙板；点击蒙板进入 VIP。
21. P50 我的页可进入记录、错题、上传、反馈、关于、登录、VIP、协议。
22. P51/P52 支持删除单条和删除全部，并有二次确认。
23. 无练习记录、无错题、无收藏、无考试记录均有空状态。

### 12.2 UI 验收

1. 页面编号与本文档一致。
2. 资料链路单独位于 ROW 04。
3. 记录标题只显示子节或试卷名称。
4. 勾选图标在目录树右侧纵向对齐。
5. 所有危险操作均有二次确认。
6. 按钮文字不溢出、不换行。
7. 题干和解析不被底部按钮遮挡。
8. 资料详情具备在线文档样式和页码。
9. VIP 资料未解锁页有磨砂蒙板。
10. 底部导航中的第三项统一为“资料”。

## 15. 总结

V1.1 以当前设计稿为准，产品主结构为“练习、考试、资料、我的”。本版本删除旧的押题密卷独立链路，新增资料中心与资料权限链路，并补齐科目切换、进度重置、目录选择、记录删除、资料下载/VIP 解锁等交互规则。

开发实现时应优先保证刷题、考试、资料和记录四条主链路完整闭环，再逐步扩展云同步、AI 评分和复杂会员能力。
