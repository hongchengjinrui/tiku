# 技术选型

## 客户端

- Flutter + Dart
- Riverpod 状态管理
- go_router 路由
- Dio 网络请求
- Drift + SQLite 本地缓存

## 后端

- NestJS + TypeScript
- PostgreSQL 主数据库
- Prisma 管理数据模型与迁移
- Swagger / OpenAPI 接口文档
- JWT + Refresh Token 鉴权

## 中台管理端

- React + TypeScript
- Vite
- Ant Design
- React Router
- Axios

## 资源与部署

- 题图、PDF、Word、封面图走 OSS + CDN
- ECS 首期跑 API、中台、Nginx 和轻量任务
- 生产数据库优先使用 RDS PostgreSQL
- 本地开发使用 Docker Compose PostgreSQL
